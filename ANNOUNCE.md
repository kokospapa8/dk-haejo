### 새 기능

- **DJ 멘트** — 곡이 바뀔 때마다 봇이 채널에 한 줄 DJ 코멘트를 자동으로 남깁니다.
- **가사 보기** — "가사 알려줘"라고 하면 현재 재생 중인 곡의 가사를 embed로 표시합니다.
- **큐 삭제 드롭다운** — 패널의 🗑️ 버튼을 누르면 드롭다운 메뉴에서 삭제할 곡을 직접 선택할 수 있습니다.
- **플레이어 재게시** — "플레이어 보여줘"라고 하면 패널이 채널 맨 아래에 다시 표시됩니다.

### 버그 수정

- 패널 자동 업데이트 안 되던 문제 수정 (repost_count 초기화 오류, 동시 edit 경합 방지)
